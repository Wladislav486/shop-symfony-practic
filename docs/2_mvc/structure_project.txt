bin - вызываемые файлы
config - настройки приложения, маршруты, сервисы
migrations - хранятся миграции
public - доступнная пользователю папка
src - фун-ал сайта
    kernel.php - работает между конфигами приложения и фун-алом
templates - шаблоны использующие шаблонизатор twig
tests - юнит тесты
translations - языковые файлы
var - файлы формирующие приложение
vendor - зависимости
.env .env.test - настройки среды
